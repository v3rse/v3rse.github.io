binder

In an attempt to get over a case of coder's block, I built myself a static generator. It's called binder. It's what generated what you're reading now.

It was inpired by stuff others have built like oscean is used to generate the XXIIVV wiki and a similar tool used to generate the Hundredrabbits' site. It aims to be a memex (memory extension) or wiki generator as the former example is.

It is something I had wanted to build for a while now. It was really fun building out a parser for a markup language, inspired to markdown, called bndr. The language spec can be found here

I look forward to write a lot more using this tool while also refining its functionality over time. It has replaced the use of hexo on this site and is aimed at being very simple.
